Using Java 21, Spring Boot 3.5.7, Maven and Angular.

To run the program:  
mvn spring-boot:run  
cd rates-angular  
ng serve

Accessible on: http://localhost:4200

Database: H2  
Link: http://localhost:8080/h2-console  
JDBC URL: jdbc:h2:mem:./data/rates  
Username: admin  
Password: admin

Task scheduling done with @Scheduled Spring annotation
